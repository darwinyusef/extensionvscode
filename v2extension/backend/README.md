# AI Goals Tracker V2 - Backend

FastAPI backend with LangGraph integration for AI-powered goal tracking.

## Features

- FastAPI REST API
- PostgreSQL with pgvector
- LangGraph for AI workflows
- Rate limiting with Redis
- Real-time WebSocket updates
- OpenAI integration

## Installation

```bash
pip install -e .
```

## Usage

```bash
uvicorn main:app --reload
```
